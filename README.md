## System requirements
We tested REAP using the following development environment:
  - Windows 11
  - Unreal Engine 5.2.1 with AirSim plugin
  - WSL2 instance (Ubuntu 22.04)
  - ROS2 Humble installed on the same WSL2 Ubuntu instance
